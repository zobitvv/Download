# **App Name**: ZobiView

## Core Features:

- Grid Menu: Display a grid menu with icons, titles, and thumbnails for each WebView.
- WebView Integration: Open WebView pages on grid item selection.
- Persistent Top Bar: Sticky top bar with app logo, name ('Zobi Tech'), and menu icons (WhatsApp contact, Notifications, WhatsApp group, Facebook profile, Share App).
- External Configuration: Admin configuration via external JSON file to add, edit, and delete WebViews.
- Conditional Content: Toggle descriptive text and ad requirement per WebView entry.  Toggle display of descriptive text and set conditional requirement to view rewarded ads before viewing.
- Launch Message: Configure launch message via JSON configuration to display at app startup.
- AI Theme Optimization: AI tool selects optimal font sizes, families, and colors based on configured theme and content.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to evoke a sense of trust and technology.
- Background color: Light gray (#F0F0F0) to provide a clean and modern backdrop.
- Accent color: Electric purple (#8E2DE2) to highlight interactive elements and add visual interest.
- Body text: 'PT Sans' (sans-serif) for body text, combining modern with warm feeling.
- Headline text: 'Space Grotesk' (sans-serif) for headlines to provide a contemporary computerized feel; use 'PT Sans' for body text.
- Use modern, flat icons for the grid menu and top bar to maintain a clean and professional look.
- Grid layout for WebViews with rounded corners and subtle shadows to create depth.
